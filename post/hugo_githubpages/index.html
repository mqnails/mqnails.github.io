用hugo创建blog

  1. 安装hugo
1

brew install hugo

这是mac上安装hugo的命令，其他平台没有安装过

  1. hugo创建站点
1

hugo new site blog
  1. 创建hugo themes
1

git clone https://github.com/rujews/maupassant-hugo themes/maupassant

这里直接选择了使用一个别人的theme

  1. 发布一个文章
1

hugo new post/xx.md

这会生成content/xx.md，修改这个markdown格式的文件作为博客文章

  1. 本机运行
1

hugo server

启动测试server后， 登录 http://localhost:1313查看效果

配置github pages

  1. 创建github pages的repository
1
2
3

登录你自己的github，比如https://github.com/mqnails
从右上角的+选择 New repository 创建一个public的repository，名字为mqnails.github.io,后缀必须为github.io
这个public的repository就是你的github pages, 域名mqnails.github.io可以对外访问

配置本机和github互信

配置ssh key使得你的github和你本机互信, 本机可以用github的ssh-url来clone/push/pull你github上的项目

  1. 本机生成ssh key
1

ssh-keygen -t ed25519 -C "mqnails@126.com"

一路回车，生成的public key在~/.ssh/id_ed25519下

  1. 在github上配置信任生成的public key
1
2
3

点击头像->Settings->SSH and GPG keys->New SSH key
随便起一个名字，把上一步生成的public key copy到key到内容中
这个时候，你在你本机上就可以通过ssh-url，免密码git clone/push/pull 你github上的项目了

配置github action token

1
2
3

点击头像->Settings->Developer settings->Personal access tokens->Tokens(classic)->Generate new token(classic)
随便起个名字， Expiration：选择No expiration， Select scopes：只勾选repo
生成的token只显示一次， 点击复制保存好

blog源码项目使用action token

配置action可以让你在提交blog github工程的时候， 执行相应的blog配置的action动作，从而可以自动编译生成静态文件，并打包发布到mqnails.github.io

1
2

进入https://github.com/mqnails/blogsource，选择Settings->Secrets and Variables->Actions->New repository secret
name随便起比如叫SECRET_FOR_PUBLISH_BLOG，Secret：填写刚才生成的action token

blog github工程配置action

在blog下新增 .github/workflows/deploy.yml内容参考如下, 如果此git push提交的话，就可以自动编译发布了

name: deploy
on:
    push:
        branches:
            - main
jobs:
    build:
        runs-on: ubuntu-latest
        steps:
        - name: Checkout
          uses: actions/checkout@v2
          with:
            submodules: true
            fetch-depth: 0
        - name: Setup Hugo
          uses: peaceiris/actions-hugo@v2
          with:
            hugo-version: "latest"
            extended: true
        - name: Build Web
          run: hugo
        - name: Deploy Web
          uses: peaceiris/actions-gh-pages@v3
          with:
            PERSONAL_TOKEN: ${{ secrets.SECRET_FOR_PUBLISH_BLOG }}
            EXTERNAL_REPOSITORY: mqnails/mqnails.github.io # 你的github pages
            PUBLISH_BRANCH: main
            PUBLISH_DIR: ./public # blog编译后的静态文件目录，它会被push到你的github pages里
            commit_message: ${{ github.event.head_commit.message }}

对maupassant主题进行定制

maupassant主题默认支持archives, about， 支持utteranc评论， 及menu，hugo.toml参考配置如下

baseURL = 'https://mqnails.github.io/'
languageCode = 'zh-CN'
title = 'mqnails'
theme = "maupassant"
summaryLength = 10
preserveTaxonomyNames = true
disablePathToLower = true
hasCJKLanguage = true
[author]
  name = "mqnails"
[params]
  author = "mqnails"
  subtitle = "On The Way..."
  keywords = "Life and Love"
  description = "Just Qing"
  busuanzi = true #启用不算子网页统计
  googleAd = ""  #ca-pub-xxxxxxxxxxxxxx
  localSearch = true # 启用本地搜索
  toc = true  # 是否开启目录
[[params.links]]
  title = "flysnow blog"
  name = "flysnow"
  url = "https://www.flysnow.org/"
[params.utteranc]
  enable = true
  repo = "mqnails/mqnails.github.io"
  issueTerm = "pathname"
  theme = "github-light"
[params.cc]
    name = "CopyLeft Free"
    link = "https://mqnails.github.io/"
[markup]
  [markup.highlight]
    lineNos = true
    style = "github"
[menu]
  [[menu.main]]
    identifier = "book"
    name = "book"
    url = "/book/"
    weight = 2
  [[menu.main]]
    identifier = "archives"
    name = "archives"
    url = "/archives/"
    weight = 3
  [[menu.main]]
    identifier = "about"
    name = "about"
    url = "/about/"
    weight = 4
